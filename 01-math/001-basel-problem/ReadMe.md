### 问题的出现

巴塞尔问题是意大利数学家蒙哥利（Pietro Mengoli）在1644年首次提出，描述所有非0自然数平方的倒数和，即：
![](./sum.png)

问题被提出后经过91年，最早被欧拉在1735年解决，因欧拉的故乡在瑞士巴塞尔，该问题也被命名为巴塞尔问题。巴塞尔问题难倒过包括牛顿在内的很多数学家，而欧拉解决这个问题时，只有28岁。
