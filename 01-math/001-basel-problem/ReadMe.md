### 问题的出现

巴塞尔问题是意大利数学家 Pietro Mengoli 在 1644 年首次提出，描述为求所有非 0 自然数平方的倒数和，即：
![](./sum.png)

问题被提出后经过 91 年，最早被欧拉在 1735 年解决，而令人意外的是结果竟然和 PI 这个无理数联系了起来。因欧拉的故乡在瑞士巴塞尔，该问题也被命名为巴塞尔问题。巴塞尔问题难倒过包括牛顿在内的很多数学家，而欧拉解决这个问题时，只有 28 岁。
